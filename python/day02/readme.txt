
1.常见的进制
    二进制
        以0b开始，由0，1组成，逢2进1
    八进制
        以0o开始
    十进制
    十六进制
        以0x开始
2.原码、反码、补码
    计算机底层是以补码的形式进行存储的
    机器数：二进制数字在计算机的表示形式
    真值： 带符号的机器数      0-正数   1-负数
    正数： 原码、反码、补码相同
    负数：
        原码：  1  + 数值绝对值
        反码：  符号位不变，其余位取反
        补码：  反码 + 1

3.涉及到的函数
    bin():将十进制转换为二进制
    oct():将十进制转换为八进制
    hex():将十进制转换为十六进制
    type(参数): 获取对应参数的数据类型
    isinstance(参数1，参数2): 判断参数1是不是参数2类型
    id():获取对象的内存地址


4.数据类型
    基本数据类型-数值
        整数int、浮点数 float、复数 complex、布尔 bool
    字符串
        str
    容器类型
        list  列表
        tuple 元组
        set   集合
        dict  字典
    特殊类型
        None

    上述类型中
        不可变的类型有：数值、字符串、元组
        可变的类型有：  列表、集合、字典
