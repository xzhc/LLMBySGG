
1.流程控制语句
    顺序
        按照程序正常的执行顺序，依次执行每条语句
    分支
        定义：分支流程又叫条件控制语句或者分支语句或者选择语句，是通过条件判断来决定执行的代码
        分类：
            单分支
                语法：
                    if 条件表达式:
                        语句1
                说明：判断条件表达式的返回值，如果返回true，执行语句1内容；如果返回false，语句1不会被执行

            双分支
                语法：
                    if 条件表达式:
                        语句1
                    else:
                        语句2
                说明：
                    判断条件表达式的返回值，如果返回true，执行语句1内容
                    如果返回false，执行语句2的内容

            多分支
                语法：
                    if 条件表达式1:
                       语句1
                    elif 条件表达式2:
                       语句2
                    elif 条件表达式3:
                       语句3
                    ...
                    else:
                注意:
                    else 至多有一个，可以没有
                    else 必须在elif语句之后
                说明:
                    判断条件表达式1的返回值，如果返回true，执行语句1内容;
                    如果返回false，执行条件表达式2的判断
                    判断条件表达式2的返回值，如果返回true，执行语句2内容;
                    如果返回false，执行条件表达式3的判断
                    如果上面所有的条件都没有满足，执行else的内容

        分支嵌套：在分支中定义分支
    match case分支
        语法：
            match x:
                case a:
                    语句1
                case b:
                    语句2
                ...
                case _:
                    语句3
       说明：
            会match后面的对象x的值和case后面的值一次进行比较
            如果x == a,执行语句1内容
            如果x == b,执行语句2内容
            如果x没有和前面的case匹配上 ,执行语句3内容
    三目运算符
        语法：表达式1 if 判断条件 else 表达式2
    循环
        在满足某个条件下，重复的执行某段代码
        while循环
            语法：
                while 条件表达式:
                    语句(循环体)
