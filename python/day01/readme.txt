5.计算机高级的运行方式
    编译型
        c、c++
    解释型
        python、javascript
    先编译再解释
        java

    注意：起始python也是先编译再解释的
         但是python的编译时机和java不一样
            java是在代码解释执行前，先进行预编译，编译后生成字节码文件落盘
            python是在对代码逐行解释执行的时候，进行的编译操作，默认生成的字节码文件在内存中

9.注释
    作用：
        给程序员看的，对程序进行解释说明
        屏蔽到掉暂时不需要的代码
        定位出错的位置
    分类：
        单行注释    #
        多行注释
            以3个引号开始，以3个引号结束

10.变量
    定义：在程序运行的过程中，其值可以改变的量
    语法： 变量名 = 变量值
    标识符命名规则：
        必须是字母、数字、下划线组成，数字不能作为开头
        不能是python中的关键字
        大小写敏感
        建议：简短 、描述性强(望文生义)
    互换变量值
        var1,var2 = var2,var1

11.常量
    定义：在程序运行的过程中，其值不会改变的量
    约定：常量名大写
