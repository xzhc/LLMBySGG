
1.模块
    定义：
        一个以.py结尾的python源文件就是一个模块
    好处：
        让代码变得更加简洁
        提供代码的复用性
        解决成员命名冲突

2.模块的导入
    全局导入
        语法：import 模块名 [as 别名]
        调用：
            如果没有起别名   模块名.成员名
            起别名          别名.成员名

    局部导入
        只导入模块的部分成员，只能使用导入的成员，没有导入的成员不能使用
        情况1：
            语法：from 模块名 import 成员名1[as 别名], 成员名2[as 别名],…
            调用：直接通过成员名调用
            对于重名成员，后导入的会将先导入的覆盖掉，可以通过别名区分不同模块的变量
        情况2：
            语法： from 模块名 import *
                  导入模块下不以单下划线开头的成员
            调用：直接通过成员名调用
            被导入的模块中使用 __all__设置哪些内容可以被导入
            __all__ 的设置只针对使用 from import * 导入模块时有效

3.__name__
    是一个特殊的内置变量
    当一个Python文件被直接运行时，该文件的__name__属性值为"__main__"。
    当一个Python文件作为模块被导入时，__name__属性会被设置为该模块的名称（即文件名，不包含 .py 后缀）。
    为了避免模块被导入的时候，测试代码被执行，可以在被导入的模块中加入__name__值的判断
        if __name__ == "__main__"
            执行测试代码

4.dir()
    dir()是内置函数
    用于列出对象的属性和方法，或者列出当前作用域中定义的名称，并以一个字符串列表的形式返回

5.包
    用来管理模块

6.导入包下模块
    全局导入
        语法：     import 包名.模块名 [as 别名]
        调用方式：  包名.模块名[别名].成员名
        使用import时，除最后一项外都必须是包。最后一项可以是模块或包，但不能是类、函数或变量
    局部导入
        局部导入包下的模块 from import
            语法: from 包名 import 模块名[as 别名]
            调用方式:模块名.成员名

        局部导入包下模块的成员 from import
            语法: from 包名.模块名 import 成员名 [as 别名]
            调用方式: 成员名

        局部导入 from import * 从包中导入模块
